## Bitsgap 是什麼？能做什麼？

Bitsgap 2017 年從愛沙尼亞起步，現在總部在杜拜，目前累計服務超過 **80 萬交易者**，跑了超過 **470 萬個機器人**，用戶託管資金超過 **94.6 億美元**。這個體量在加密自動交易領域算得上是老牌玩家了。

核心功能大概分幾塊：

**🤖 6 種交易機器人**

不同市場狀況用不同機器人，這是 Bitsgap 比較特別的設計：

- **GRID Bot**：橫盤市場的最愛，在設定的價格區間內自動高賣低買，網格策略。平台數據顯示，GRID Bot 近 30 天平均收益約 11%。
- **DCA Bot**：分批建倉，降低平均成本，適合震盪或緩慢趨勢市場。
- **BTD Bot**（Buy The Dip）：下跌行情裡低接做多，等市場反彈後獲利。
- **LOOP Bot**：長期持倉策略，能同時在兩種幣種裡賺錢，利潤自動複投。
- **DCA Futures Bot**：期貨市場的 DCA 策略。
- **COMBO Bot**：結合 GRID + DCA，適合期貨市場的高風險高收益玩法。

**📊 統一交易終端**

現貨和期貨都能操作，還整合了 TradingView 圖表，支持 TWAP、止損限價、Trailing Take Profit 等高階掛單方式。

**🧠 AI 助理**

根據你的資金量和風險偏好，AI 會給出個性化的機器人設定建議，PRO 方案還有 AI Portfolio Mode 幫你一鍵配置多個機器人組合。

**🔒 資金安全**

Bitsgap 透過 API 連接你的交易所帳戶，**它無法提領你的資金**，也不存放你的幣。從 2017 年至今，沒有發生過安全事故，API 金鑰採用 2048 位加密保護。

**適合你如果：**
- 你有多個交易所帳戶，想統一管理
